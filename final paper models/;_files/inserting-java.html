Insert Multiple Documents using the MongoDB Driver for Java

In this activity, you will create a method that inserts multiple documents into a collection using the MongoDB Driver for Java. You will use this method to add multiple documents to the accounts collection for the bank database using a provided array of documents called sampleDocuments.

Lab Instructions

  1. Open the Crud.java file in the IDE by clicking the file name in the file explorer to the left.

  2. Create an insertManyDocuments method such that it accepts an array of documents as an argument and inserts it in the accounts collection within the bank database.

  3. The documents to insert are defined in the sampleDocuments variable. This can be passed to the insertManyDocuments function as an argument. Print the number of inserted documents.

  4. When you are ready to run your code, click the floppy disk icon next to the file name to save. Then, select the terminal tab at the top of the screen and navigate to your project's root folder.

  5. Compile your Maven project by running:

    mvn compile

  6. Run the following command:

    mvn exec:java -Dexec.mainClass=com.mdbu.app.DemoApp

Once you complete the lab, compare your answer to the correct answer in the Review and Solved Code tab, then select Next.

  • To add a new document to a collection, you will need to use the insertMany method on the collection object like so: db.accounts.insertMany(documents)

  • The function is already invoked in the DemoApp.java file, so all you need to do is add logic for the insertManyDocuments function, and run the application.

  • insertMany is a method that accepts an array of documents as an argument, and inserts them all into the collection. The function returns an InsertManyResult object that contains information about the documents inserted.

  • The MongoCollection object has already been initialized to simplify the code.

Let's create an empty method called insertManyDocuments in Crud.java:

public void insertManyDocuments() {

}

Then, let's pass in the multiple documents to insert as an array and use the insertMany method to insert them:

public void insertManyDocuments(List<Document> documents) {
  InsertManyResult result = collection.insertMany(documents);
  System.out.println("\tTotal # of documents: " + result.getInsertedIds().size());
}

Then, compile your code:

mvn compile

Run the following command:

mvn exec:java -Dexec.mainClass=com.mdbu.app.DemoApp
